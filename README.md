📌 Project Overview

This project analyzes retail sales performance using the Blinkit dataset.

An interactive Power BI dashboard was developed to evaluate outlet performance, product distribution, and revenue contribution across different outlet tiers and categories.

The objective was to transform raw retail data into actionable business insights through KPI tracking and visual analytics.

🎯 Business Objectives

Track overall sales performance and product distribution

Analyze revenue contribution by outlet tier and size

Identify top-performing product categories

Evaluate customer ratings and sales impact

Enable dynamic filtering for deeper business analysis

📈 Key Metrics & KPIs

Total Sales: $1.2M+

Number of Items Analyzed: 8,500+

Average Sales per Item

Average Customer Rating

Outlet Tier-wise Revenue Contribution

🛠 Tools & Technologies Used

Power BI – Dashboard development & data modeling

Microsoft Excel – Data cleaning & preprocessing

SQL – Data querying and transformation

⚙️ Data Preparation Process

Cleaned and validated retail dataset

Handled missing values and standardized categories

Created calculated measures for KPIs

Built data model relationships in Power BI

Designed performance-optimized visuals

📊 Dashboard Features

KPI summary cards

Tier-wise and outlet-size revenue analysis

Category-level sales breakdown

Fat-content-based product comparison

Interactive slicers for dynamic filtering

🚀 Key Insights

Tier 3 outlets contributed the highest revenue share

Certain product categories consistently outperformed others

Sales distribution varied significantly across outlet sizes

Identified peak establishment year contributing to revenue growth
